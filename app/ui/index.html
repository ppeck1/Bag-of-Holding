Score formula reference ▶
Formula
0.6 × text + 0.2 × canon + 0.2 × planar + conflict_penalty + daenary_adjustment
text_score
Normalized BM25 (FTS5) → [0,1]
canon_score
canon_score(doc) / 180 clamped to [0,1]
planar
Weighted direction sum over active plane facts → [0,1]
conflict_penalty
−0.1 if doc has any open conflict
daenary_adjustment
Additive only, range [−0.05, +0.05]. High-confidence dims +0.05, stale dims −0.05.

Mutation Safety Rules
✓ Imported files are draft/reference by default — never auto-canonical.
✓ LLM suggestions never become canonical without explicit user approval.
✓ Canonical status requires a user workflow action (release state).
✓ Every lifecycle change is recorded in the lifecycle history log.
✓ Backward movement and undo are available in the document drawer.
✓ All mutations are appended to the audit log — never silently erased.
