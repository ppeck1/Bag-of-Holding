Known-Good HTML Import Fixture

Phase 26.2 Verification Document

Purpose

This document verifies that BOH correctly processes HTML files during import. The importer should:

  • Extract the title from the <title> tag
  • Strip navigation, header, footer, scripts, and styles
  • Extract headings as Markdown-style headers for topic indexing
  • Preserve body text for search
  • Assign draft status on import

Content for Indexing

This section contains body text that should be preserved and indexed. The HTML importer must normalize this into readable text without raw tag markup appearing in search results.

Key Facts

Conclusion

If the title, headings, and body text are correctly extracted and indexed, the HTML import pipeline is functioning correctly.
