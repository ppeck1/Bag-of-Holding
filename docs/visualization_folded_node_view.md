# Visualization And Folded Node View

The visualization surfaces expose persisted knowledge state without changing it.

## Graph And Fold Views

- Web view emphasizes topology and relationships.
- Evidence State emphasizes evidence and plane state.
- Risk Map emphasizes drift and readiness pressure.
- Authority Path emphasizes authority lanes.
- Timeline emphasizes temporal position.

Folded-node packets summarize a selected document with scalar metrics, symbolic
state, unresolved unknowns, and navigation affordances. The packet is read-only.

## UI Rule

Visualization may help an operator decide what to review next. It must not
promote, delete, normalize, or resolve knowledge automatically.
